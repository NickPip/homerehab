# Public Assets

This folder contains static assets that are served directly by Next.js.

## Logo

Please add your HomeRehab logo file here as `logo.png` or `logo.svg`.

The logo should be:
- PNG or SVG format
- Transparent background (recommended)
- High resolution for crisp display
- Optimized file size

The Header component will automatically use `/logo.png` or you can update the path in `app/components/Header.tsx` if you use a different filename.
